A sleek and fully responsive food delivery web UI featuring clean layouts, reusable components, and dark/light theme support. Designed to deliver a polished user experience across all devices.

## Features

* Built with Next.js for performance and routing
* Tailwind CSS for fast, utility-first styling
* Dark and light theme support
* Fully responsive across mobile, tablet, and desktop
* Modular and reusable components

## Tech Stack

* Next.js
* TypeScript
* Tailwind CSS
* React Icons

## Getting Started

```bash
git clone https://github.com/your-username/foodie.git
cd foodie
npm install
npm run dev
```
